# Product Overview

## What is 7D GenAI UI?

7D GenAI UI is a **Python project management and development platform** that provides a unified workspace for creating, managing, and editing Python projects. It combines template-based project scaffolding with visual ETL workflow editing capabilities.

## Target Users

- **Data Engineers** - Building and maintaining ETL pipelines
- **ML Engineers** - Creating data processing workflows for ML pipelines
- **Python Developers** - Managing Python packages and APIs
- **Analytics Teams** - Working with workflow orchestration tools

## Core Value Proposition

A single platform where users can:
1. Create Python projects from pre-built templates
2. Import existing projects from various sources
3. Edit ETL workflows visually (using React Flow) or via code
4. Manage multiple projects within organized workspaces

## Key Differentiators

### Template-Based Project Creation
- 8 project templates across 5 categories
- Consistent project structure and best practices
- Quick start for common Python project types

### Multi-Tool ETL Support
- Support for 4 major workflow orchestrators:
  - Dagster, Prefect, Dagu, Airflow
- Visual DAG editing with React Flow (planned)
- Unified interface regardless of underlying tool

### Project Import Capabilities
- Import from Data Guru format (convert to any supported ETL tool)
- Import existing UV Python projects
- Import Python packages

### Workspace Organization
- Group related projects together
- Cross-project dependencies (planned)
- Centralized project management
